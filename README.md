# Socketeria
C++ crossplatform socket (tested on windows 8 and GNU\linux)

## Dependencies
* Any compiler that supports gnu++11/c++11 (Tested on GCC>=4.7 and MinGW)
* CMAKE and MAKE for compiling (>=2.6)
* Doxygen for creating the documentation (>=1.8.6)

## Compiling

```bash
mkdir build
cd build/
cmake..
make
```
